🚀 FitData 可视化工具 - 离线使用指南

✅ 恭喜！ 您已成功获得完整的本地部署包，可以完全离线使用此工具。

📋 快速启动步骤

步骤 1: 确保已安装 Node.js
  • 如未安装，请访问 https://nodejs.org/ 下载安装
  • 验证安装：打开命令行输入 node --version
步骤 2: 选择启动方式
  • Windows 用户: 双击 start-local.bat 文件
  • PowerShell 用户: 右键选择"用PowerShell运行" start-local.ps1
  • 手动启动: 在 fitdata-visualizer 目录下运行 npm start
步骤 3: 等待自动启动
  • 首次运行会自动安装依赖（需要联网）
  • 后续使用完全离线
  • 浏览器将自动打开 http://localhost:3000

🎯 主要功能

  • 数据导入: 支持 Excel (.xlsx) 和 CSV 文件
  • 可视化: 交互式热力图显示
  • 预测分析: 多种数学拟合算法
  • 实时交互: 双击查看详细预测结果

📊 使用方法

  1. 启动应用后，点击"选择文件"按钮
  2. 选择您的数据文件（支持 .xlsx 或 .csv 格式）
  3. 等待数据加载和可视化生成
  4. 双击热力图中的任意数据点查看预测结果
  5. 查看包含多种算法的详细拟合分析
⚠️ 重要提醒:
  • 首次运行需要联网下载依赖包
  • 依赖安装完成后可完全离线使用
  • 建议使用 Chrome、Firefox 或 Edge 浏览器

🔧 故障排除

问题: 双击启动脚本没有反应
解决: 检查 Node.js 是否正确安装，运行 node --version 验证
问题: 提示"npm 不是内部或外部命令"
解决: 重新安装 Node.js，确保添加到系统 PATH
问题: 端口 3000 被占用
解决: 关闭其他占用 3000 端口的程序，或重启计算机
